## project_1

**Version 1.0.0**

#INTRODUCTION

The project analyses top 30 + coins from Coin Market Cap to run a 5-year historical analysis.  

#REQUIREMENTS
Windows 10 OS or Mac OS

## Below modules are needed to run complete analyses: 
-	OS
-	dotenv
-	Pandas
-	Pandas_datareader
-	Numpy
-	Hvplot
-	McForecastTools
-	Alpaca_trade_api
-	Sqlalchemy
-	Matplotlib
-	pathlib
